Blizzard API data is collected and stored in the DB. We aggregate the data, index it, and derive useful statistics from it. This data is displayed on different platforms such as our website and Discord bot to improve experience of players.

Blizzard data retention

We strictly follow the Blizzard ToS, which forces us to permanently delete the data within 30 days after it was deleted from the Blizzard's API. Bear in mind that StartCraft®2 leaderboards are deemed as public data(section 15). You can remove some of your data by submitting a data removal request to Blizzard, disabling 3rd party access in settings of your Blizzard account, or using the BattleNet data manager. We update affected player data of the last ranked ladder seasons, other data may be removed when it expires unless it's updated from other seasons.

Revealed/Unmasked accounts

Account revealing is based on public data and will not be removed. We usually remove some info if users request it, but this is a special case because account removal will trigger a chain reaction from other users and the whole feature will be useless.
